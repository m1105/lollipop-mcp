# 03 — 管理員日常操作

> Last updated: 2026-04-06

這份文件是針對非技術背景的 bot 管理員。你不需要知道工具名稱——只要用自然語言告訴 Claude，Claude 會自動執行正確的 MCP 命令。

---

## 你需要準備

- 已完成 [01 — Lollipop Plugin 安裝及連線設定](01-install-plugin.md)
- 至少一隻 bot 在線運行中（可用 `bot_list` 檢查）

---

## 操作速查表

| 我想要... | 怎麼說 | 系統做什麼 | 備註 |
|-----------|--------|-----------|------|
| 看整體狀態 | "報告" / "狀態" / "早班報告" | workshop_morning_report | 含機台+bot+排程+警告 |
| 誰缺補給 | "誰缺補給" / "補給狀態" | fleet_supply_check | 檢查所有 bot 的箭、藥等 |
| 有人死了嗎 | "誰死了" / "健康檢查" | fleet_health_check | 標記 HP<30% 或無法操作的 |
| 搬 bot 到地點 | "把小明移到古魯丁" | bot_teleport + setup + start | 需說角色名+地點 |
| 今天賺多少 | "今天賺多少" / "績效" / "賺錢" | fleet_performance | 預設 1 天，可加"這週"看 7 天 |
| 全部停止 | "全停" / "緊急停止" | bot_all_stop | 所有機器的所有 bot |
| 重啟某台 | "小明重啟" / "重開煎餅狗子" | bot_stop + bot_start | 重新開始戰鬥 |
| 加新帳號 | "加新帳號 XXX" | opener_add_account | 需提供帳密、伺服器號 |
| 排班 | "幫我排明天的班" | schedule_templates → generate | 需先建立模板或說明班表 |
| 補給觸發 | "現在補給" / "馬上補給" | bot_supply_trigger | 立即執行補給流程 |
| 緊急逃離 | "小明逃出來" | bot_setup_protect + escape | 低血量自動逃離設定 |

### Demo 1: 早班報告（Morning Report）

**你**: 報告

[Claude 呼叫 `workshop_morning_report`]

**Claude**: 
```

⚠ 大熊貓 (Lv35 妖精) — 說話之島，12/40 HP，CRITICAL！

1. 大熊貓 HP 過低 — 需補血或修理裝備
2. 破碎的夢 等待復活
```

**你**: 大熊貓什麼情況？為什麼 HP 這麼低？

[Claude 呼叫 `bot_check_health` 針對大熊貓]

**Claude**:
```

- HP: 12/40 (30%)
- 背包: 治療藥水 x2，魔力恢復藥水 x0
```

[Claude 順序執行:

3. interact 溫諾，修理身體防具
4. 返回掛機點
5. `bot_start`]

**Claude**:

## 進階技巧

### 組合指令

你可以組合多個命令，Claude 會按順序執行：

**你**: 報告一下，順便補給缺的，然後告訴我今天賺多少

Claude 會:
1. 執行 `workshop_morning_report`
2. 檢查 `fleet_supply_check`
3. 觸發補給 `bot_supply_trigger`
4. 查詢 `fleet_performance`

---

### 批次操作

針對特定地點的 bot：

**你**: 把所有在古魯丁的 bot 換到說話之島

Claude 會:
1. 呼叫 `bot_list` 篩選位置 = 古魯丁
2. 逐台執行傳送 + 設置 + 重啟

---

### 時間範圍

**你**: 這週的績效

Claude 理解 "這週" = 7 天，呼叫 `fleet_performance(days=7)`

**你**: 昨天的報告

Claude 會查詢過去 24 小時的數據

---

### 職業特定操作

某些職業可用的傳送點不同。Claude 會自動根據職業選擇正確的目的地：

**你**: 把妖精都傳送到妖精森林村

Claude 會:
1. 篩選職業 = 妖精
2. 檢查是否有妖精森林村傳送點（妖精才有）
3. 執行傳送 `bot_teleport_scroll("妖精森林村|...")`

---

### 緊急逃離

設定低血量自動逃離：

**你**: 小明血量低於 20% 要逃離

[Claude 呼叫 `bot_setup_protect`]

```
escape_enabled = 1
escape_hp_pct = 20
escape_items = ["說話的卷軸", "回城卷軸"]
```

之後小明自動偵測血量，低於 20% 時自動傳送到安全地點。

## 常見問題

### Q: 我想要某個 bot 停止掛機，但保留其他的

**A**: 說出角色名：

**你**: 停止小狐狸

Claude 會單獨停止小狐狸，其他 bot 繼續運行。

### Q: Bot 卡住了，怎麼辦？

**A**: 說：

**你**: 小狐狸卡住了，重啟

Claude 會執行 `bot_stop` 然後 `bot_start`，通常能解決。

### Q: 我想看某個 bot 的詳細狀態

**A**: 說角色名：

**你**: 煎餅狗子狀態

Claude 會呼叫 `bot_status` 顯示:
- 位置 (x, y)
- HP / MP
- 背包物品 + 數量
- 運行狀態

### Q: 想知道還能掛多久

**A**: 說：

**你**: 還能掛多久

Claude 會根據當前:
- 背包容量
- 補給狀態
- 掛機速度
預估掛機時間。

### Q: 排班後想臨時改時間

**A**: 說具體的改動：

**你**: 明天小狐狸改成 09:00 開始而不是 08:00

Claude 會呼叫 `schedule_update` 修改單條排程。

## 快速參考

### 最常用的 5 個命令

1. **報告** — 一眼看完所有狀態
2. **誰缺** — 檢查補給需求
3. **誰死** — 找出有問題的 bot
4. **績效** — 看收益
5. **全停** — 緊急停機

### 回應時間預期

| 操作 | 時間 |
|------|------|
| 報告 | 2-3 秒 |
| 檢查補給 | 3-5 秒 |
| 搬 bot | 30-60 秒 |
| 補給流程 | 3-5 分鐘 |
| 排班生成 | 5-10 秒 |
