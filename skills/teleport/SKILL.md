# 傳送系統

## 說話的卷軸（F12）

`bot_teleport_scroll(dest="村莊|傳送點")`

- **不要用 `bot_useitem()`** — 會觸發隨機傳送
- 正確流程：scroll_show → scroll_dest

### 目的地格式

`"村莊|傳送點"`，例如 `"說話之島|雜貨商人"`

### 通用目的地
- 說話之島: 旅館, 倉庫管理員, 傳送師, 雜貨商人, 魔法書商人, 說話之島地監入口, 北島
- 古魯丁村: 旅館, 倉庫管理員, 傳送師, 雜貨商人, 古魯丁地監入口
- 奇岩村: 倉庫管理員, 傳送師, 雜貨商人, 武器商人, 防具商人, 藥水商人, 奇岩地監入口
- 共同: 正義神殿, 邪惡神殿, 沙漠綠洲, 祝福之地

## NPC 傳送師

`bot_teleport_npc(dest="海音", scroll_dest="奇岩村|傳送師")`

流程：scroll 傳送到傳送師 → interact → 「想去其他地區」→ 選目的地

| 村莊 | 傳送師 | 座標 |
|------|--------|------|
| 說話之島 | 盧卡斯 | (32580,32929) |
| 古魯丁村 | 史提夫 | (32611,32732) |
| 奇岩村 | 爾瑪 | (33437,32798) |

## 祝福卷軸（F11）

`bot_teleport_blessed(dest)` — 用 bot_bookmarks 查書籤列表，bot_bm_click(idx) 傳送。
